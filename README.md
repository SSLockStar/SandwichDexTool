# SandwichTools

## Backend - API Server
## Installation

```bash
$ npm install
```

## Settings
### .env
```bash
PORT=4000
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
